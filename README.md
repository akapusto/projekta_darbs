Projektu sadalīju uz vairākiem failiem, kuri dara atsevišķas lietas

download_books.py lejupielade lielu masīvu ar .txt grāmatam no saites www.gutenberg.org 

count_frequencies.py analizē lejupieladētas grāmatas uz frekvencem

normalize_frequencies.py notīra saņemtu pēc count_frequencies.py tabulu no liekiem simboliem

encrypt.py - programma, kas ļauj (de)šifrēt .txt failu ar zināmu atslēgu. Šī projekta daļa ir nepieciešama, lai pēc tam būtu ko dešifrēt

decryptor.py - programma, kura dešifrē iešifrēto ar Cēzara vai Viženera algoritmu tekstu(angļu valodā)
